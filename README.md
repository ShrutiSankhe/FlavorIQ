# FlavorIQ

FlavorIQ is a 0?1 flavor-intelligence product that helps users understand ingredient flavor profiles and make confident pairing decisions using structured sensory data and explainable reasoning.

## Problem
Home cooks often know ingredients individually but lack a shared language for understanding why flavors work together, making substitutions and creative cooking difficult.

## Solution
FlavorIQ maps ingredients across a controlled flavor taxonomy (aroma, intensity, perceived temperature, regions, techniques) and evaluates ingredient compatibility using explicit pairing heuristics.

## MVP Scope
- Ingredient flavor cards
- Pantry-aware pairing evaluation
- Explainable ?why it works / why it doesn?t? logic

## Non-Goals (v1)
- Full recipes
- Nutrition tracking
- User accounts
- Social features
